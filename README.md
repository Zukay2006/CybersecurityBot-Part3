# Cybersecurity Awareness Chatbot – Part 3

## Project Overview

This project is a Cybersecurity Awareness Chatbot developed in C# using Windows Presentation Foundation (WPF). The chatbot educates users about common cybersecurity threats while providing an interactive and engaging experience through quizzes, sentiment detection, memory features, and random cybersecurity tips.

## Features

* Interactive chatbot conversation.
* Cybersecurity awareness tips.
* Snake Roll Quiz game.
* Quiz scoring system.
* Dice feature for random activities.
* Memory system that remembers user information during the session.
* Sentiment detection to provide appropriate responses.
* Keyword recognition for cybersecurity topics.
* Audio welcome message.
* User-friendly WPF graphical interface.

## Technologies Used

* C#
* Windows Presentation Foundation (WPF)
* .NET
* Visual Studio

## Setup Instructions

1. Download or clone this repository.
2. Open the solution (.sln/.slnx) in Visual Studio.
3. Restore any required NuGet packages if prompted.
4. Build the solution.
5. Run the project.

## How to Use

1. Launch the application.
2. Enter your name when prompted.
3. Ask cybersecurity-related questions.
4. Complete the quiz to test your knowledge.
5. Explore the chatbot's different features and responses.

## Example Interaction

**User:** What is phishing?

**Chatbot:** Phishing is a cyberattack where attackers attempt to trick users into revealing sensitive information through fake emails, messages, or websites.

**User:** Give me a cybersecurity tip.

**Chatbot:** Always use strong, unique passwords and enable two-factor authentication whenever possible.
